# Acceptance Criteria

Use these criteria to check whether a change preserves the existing DataInsight Agent product surface.

- AC-DATA-01: Dataset preview tabs remain visible: 前20行, 后20行, 随机20行, 字段信息.
- AC-DATA-02: Selecting uploaded, appended, cleaned, and joined datasets should not crash the workspace.
- AC-DQ-01: Data quality center keeps missing value, duplicate, ID, outlier, and IQR sections.
- AC-UI-01: Switching main tabs should not create bottom error blocks.
- AC-EXPORT-01: Report export center remains available.
- AC-NO-DATA-01: Empty or no current dataset state shows friendly messages instead of uncaught exceptions.
